# Static Files Directory

This directory contains static assets for the Android Log Capturer web application.

## Required Files:

1. **favicon.ico** - Website favicon
2. **netcore_logo.svg** or **netcore_logo.png** - NetCore company logo

## Current Files:

- **netcore_logo.svg** - Placeholder NetCore logo (replace with your actual logo)

## Instructions:

Please place your favicon.ico and actual NetCore logo files in this directory.

The application will automatically serve these files from:
- `/static/favicon.ico` (for the browser tab icon)
- `/static/netcore_logo.svg` (for the header logo)

**Note:** The current logo is a placeholder. Replace it with your actual NetCore logo file.
